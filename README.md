# E-commerce API (Backend)

This is a backend-only **E-commerce API** developed to handle core e-commerce operations.  
The project provides RESTful APIs with complete **CRUD operations** using **PostgreSQL** as the database.

---

# Features

- RESTful API architecture
- CRUD Operations (Create, Read, Update, Delete)
- PostgreSQL Database Integration
- Clean and structured backend code

---

# Technologies Used

- Backend Language / Framework
- REST API
- PostgreSQL
- VS Code

---

# Project Type

- Backend Only
- API Based Project

---

# Database

- PostgreSQL
- Used to store e-commerce related data (products, users, orders, etc.)
- Database operations handled using CRUD APIs
